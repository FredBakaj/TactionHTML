Вступить по ссылке

UMLUseCase
TAction :: Юзер кейсы :: Вступить по ссылке

Description

Эта фича подразумевает что пользователи могут по ссылке вступить в сообщество. Эту ссылку генерирует Админ сообщества, после чего отправляет всем желающим кото хочет вступить.

Формат ссылки для публичного сообщества. app-domain.com/name-comunity

Формат для приватного сообщества. app-domain.com/23j1o120
