UMLActor
TAction :: Use Cases :: Actors :: Создатель шаблона КД

Description

none

Use Cases

  • Создание собственного шаблона челенджа
  • Редактирование пользовательского шаблона (в ограниченном виде)
  • Копирование существующего шаблона
  • Удаление пользовательского шаблона
  • Автоматическая генерация обложки челенджа (AI)
  • Подсказки при создании шаблона
  • (СТАТ) Статистика по шаблону (кол-во запусков, завершений)
  • Получение бейджей за популярные шаблоны
  • Создание расширенного шаблона
  • Возможность тестирования шаблона перед публикацией
  • Поддержка логики «бота» внутри шаблона
  • Приватный токен доступа для подписки на события
  • Отладка шаблона через песочницу
  • Управление разрешениями шаблона (например, кто может запускать)
  • (СТАТ) Поддержка групповой статистики по шаблону
  • Я хочу регистрировать вебхук
  • Добавить новую локализацию в шаблон
