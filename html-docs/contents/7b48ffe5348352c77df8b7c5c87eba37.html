Извлечь n количество КД по каталогу

BPMNTask
TAction :: BPMN Old :: BPMN Выбрать КД из Каталога :: Api :: Извлечь n количество КД по каталогу

Description

Извлечь записи КД из базы для получения актуального каталога. Эта задача также учитывает фильтры, и сортировку. Чтобы можно было при повторном открытии каталога отправить повторно запрос.
